# Tracking Spreadsheet Setup — Test Dental

## Google Sheets Setup
1. Create a new Google Sheet titled "Test Dental — Automation Tracker"
2. Create these columns in Row 1:

| A: Patient Name | B: Phone | C: Email | D: Last Visit | E: Recall Due | F: Missed Call | G: Follow-up Sent | H: Appt Booked | I: Notes |

## Color Coding
- Red: Recall overdue (>7 months since last visit)
- Yellow: Recall due this month
- Green: Recently seen (within 3 months)

## Monthly Review Checklist
- [ ] Count new missed calls recovered this month
- [ ] Count no-shows vs. prior month
- [ ] Count reactivated patients
- [ ] Note any replies to automation messages
- [ ] Update "Last Visit" dates for seen patients
